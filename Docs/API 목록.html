4. WebLog

다음과 같은 URL로 구성됩니다.

/api/v1/weblog

공통으로 쓰이는 Parameter 인 interval은 다음과 같습니다.

"1m", "5m", "10m", "30m", "1h", "6h", "12h", "1d", "1w", "1mo", "6mo", "1y"

(형식에 맞지 않을 경우 400상태코드가 반환됩니다.)

Weblog

이름HeaderRequest TypeBody Type설명
🧭/componentsNoneGETNone컴포넌트 집계 기록을 반환한다.
🧭/clicksNoneGETNone클릭 집계 기록 정보를 반환한다.
🧭/durationsNoneGETNone쳬류시간 집계 기록 정보를 반환한다.
🧭/journalsNoneGETNone페이지 이동 집계 기록 정보를 반환한다.
🧭/refersNoneGETNone페이지 유입경로 집계 기록 정보를 반환한다.
